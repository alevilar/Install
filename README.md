Instrucciones
=======

1) - Borrar el database.php del /Config
2) - Reemplazar el bootstrap.php del /Config por la version que esta ubicada en Install/Config/Docs/bootstrap.php
3) - Remplazar el AppController.php de la carpeta /Controller por la version ubicada en Install/Config/Docs/
4) - Remplazar el AppModel.php de la carpeta /Model por la version ubicada en Install/Config/Docs/ 
Nota: El cambio detallado en el punto 2 es para detectar si esta o no instalado el Ristorantino, de no estar instalado se invoca al /install para iniciar el proceso de instalación.
